# 响应式

对于对象可以使用 递归 来响应式化，但是数组我们也需要处理

- push
- pop
- shift
- unshift
- reverse
- sort
- splice

要做什么事情呢？

1. 在改变数组数据的时候，发送通知
    * vue 2 中缺陷，设置 length 无法发送通知（Vue3 使用 Proxy 语法 es6 解决了）
2. 加入元素应该变成响应式的

技巧：如果一个函数已经定义了，但是我们需要扩展其功能，我们一般的办法：

步骤：（参考 03.html ）

1. 使用一个临时的函数名存储函数
2. 重新定义原来的函数
3. 定义扩展的功能
4. 调用临时的那个函数


扩展数组的 push 和 pop 怎么处理？？？

方案：修改要进行响应式化的数组的原型（ __proto__ ）
